# Role & Objective
Act as a Senior Frontend Architect. Your goal is to set up the foundational HTML and CSS structure for the project. Do NOT generate content yet, only the skeleton.

# Structure & Architecture Rules
1. **Tech Stack:** Use strictly plain HTML5, CSS3, and modern JavaScript (ES6+). No frameworks (React/Vue/Bootstrap).
2. **Semantic HTML5:** Strictly divide the layout into three main areas:
   - `<header>` for navigation.
   - `<main>` to wrap all future page content.
   - `<footer>` for the bottom area.
3. **CSS Foundation:** - Create a `:root` selector with CSS Variables for primary colors, secondary colors, text colors, and fonts.
   - Create a global `.container` class to align content centrally across all sections.
4. **Code Organization:** Add clear comments in HTML and CSS (e.g., `<!-- Header Section -->`) to visually separate blocks.
