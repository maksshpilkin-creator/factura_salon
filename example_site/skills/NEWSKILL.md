# Kikimora Studio – Visual System

## 🎨 Brand Color System (STRICT)

Use ONLY these colors:

:root {
  --color-primary: #F26A1B;        /* Main orange from chairs */
  --color-primary-soft: #FFA365;   /* Soft orange accent */
  --color-bg-main: #FFF6EE;        /* Warm beige background */
  --color-bg-white: #FFFFFF;
  --color-wood: #6B3E2E;           /* Dark wood panels */
  --color-text-main: #2B2B2B;
  --color-text-muted: #6E6E6E;
  --color-border: #F0E3D8;
}

❗ No blue.
❗ No cold gradients.
❗ No neon colors.
❗ No generic startup UI palette.

Design must feel:
- Warm
- Clean
- Bright
- Feminine but modern
- Premium salon atmosphere

---

# 🪑 Style Rules

## Interior Inspired Design
- Rounded corners: 16px–24px
- Soft shadows (like mirror lighting)
- Cards should feel like salon furniture blocks
- Use warm light glows, not sharp shadows

Example:
box-shadow: 0 10px 40px rgba(242, 106, 27, 0.08);

---

# 🔤 Typography

Use:
- Headings: Playfair Display or Cormorant
- Body: Inter

Headlines should feel editorial, like beauty magazine.

Example:
"Hairs that speak before you do"

Avoid robotic AI copy.

---

# 🧱 Layout Rules

- Large breathing space
- Grid based layout
- No clutter
- Big imagery blocks
- Accent sections with orange background

Alternate sections:
1. Warm beige
2. White
3. Soft orange

---

# ✨ Animation Style

Subtle only:
- Fade in
- Slight lift on hover
- Glow on buttons

Buttons:
background: var(--color-primary);
border-radius: 999px;

Hover:
transform: translateY(-3px);
box-shadow: 0 12px 30px rgba(242,106,27,0.25);

---

# 📸 Image Style

Photos must:
- Show interior
- Show warm lighting
- Show orange chairs
- Show mirrors with bulbs
- Avoid stock photos
