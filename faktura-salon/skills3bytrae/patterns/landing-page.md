# Landing Page Pattern

The standard high-converting structure for service businesses.

## Section Order

1.  **Header/Nav**: Sticky. Logo left, Links right, CTA button.
2.  **Hero**: Value proposition + Primary CTA.
3.  **Trust Bar**: Logos of brands used, certifications, or "As seen in".
4.  **Problem/Agitation**: "Struggling with [X]?" (Optional, depends on niche).
5.  **Solution/Services**: 3 main services highlighted.
6.  **Value Proposition/Features**: Why us? (Licensed, Insured, Fast).
7.  **Social Proof**: Testimonials/Reviews.
8.  **Portfolio/Gallery**: Visual proof of work.
9.  **FAQ**: Address objections.
10. **Final CTA**: "Ready to start?"
11. **Footer**: NAP, Links, Copyright.

## Layout Notes
-   Alternating background colors (White -> Light Grey -> White) to break up sections.
-   Consistent vertical spacing (`var(--spacing-xl)`).
