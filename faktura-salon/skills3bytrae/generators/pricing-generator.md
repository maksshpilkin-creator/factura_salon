# Pricing Table Generator

Use this prompt to generate clear pricing options.

## Prompt

```markdown
Create a 3-column pricing table for a [Niche] business.

**Columns:**
1. **Basic**: "Essential" - For small needs.
2. **Standard**: "Most Popular" - The sweet spot (highlight this one).
3. **Premium**: "All-Inclusive" - For VIP treatment.

**HTML Structure:**
- Use a CSS Grid layout.
- The "Standard" card should have a `border: 2px solid var(--color-primary)` and a "Popular" badge.
- List features using `<ul>` with checkmark icons.
- Each card must have a CTA button.

**Copy Rules:**
- Avoid "Contact us" for prices if possible. Use realistic ranges (e.g., "From $99").
- Highlight the value, not just the cost.
```
