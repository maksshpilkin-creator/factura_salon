# Generic Section Generator

Use this prompt to build a custom section (e.g., "About Us", "Our Process").

## Prompt

```markdown
Create a responsive HTML/CSS section for "[Section Topic]".

**Layout Options (Choose best fit):**
1. **Split**: Text on left, Image on right.
2. **Center**: Text centered, max-width 800px.
3. **Grid**: Cards or icons in a responsive grid.

**Requirements:**
- Use semantic tags (`<section>`, `<h2>`, `<p>`).
- Use CSS variables from the Design System.
- Ensure mobile responsiveness (stack on small screens).
- Add "Scroll Reveal" animation class if applicable.
```
