# Content Rules & Copywriting

This file defines the voice and tone for all local business landing pages. The goal is "Premium Local Business" - trustworthy, professional, and high-quality.

## Voice & Tone

-   **Professional but Approachable**: Speak like a helpful expert (e.g., a skilled dentist, a master carpenter).
-   **Local-First**: Mention the city/neighborhood. Connect with the local community.
-   **Direct & Clear**: Avoid jargon. State the value proposition immediately.

## Banned Words (Strictly Prohibited)

Never use these words in generated content, code comments, or commit messages. They sound like generic AI.

-   seamless
-   performant
-   mission-critical
-   leverage
-   synergy
-   cutting-edge
-   innovative
-   robust
-   scalable
-   "it's not just X it's Y"
-   world-class

## Replacement Guide

| Avoid | Use Instead |
| :--- | :--- |
| **Seamless** | Easy, simple, quick, hassle-free |
| **Innovative** | Modern, new, updated, smart |
| **Cutting-edge** | Advanced, latest technology, premium |
| **Leverage** | Use, build on, take advantage of |
| **World-class** | Expert, top-rated, professional |

## Formatting

-   **Headings**: Title Case (e.g., "Our Premium Services").
-   **Body Text**: Sentence case.
-   **Paragraphs**: Max 3 lines. Short and punchy.
-   **Lists**: Use bullet points for benefits.
