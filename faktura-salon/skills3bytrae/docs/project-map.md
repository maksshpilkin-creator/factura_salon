# Project Map

This project is an AI website generation system.

The architecture follows layered design.

---

# Layer 1 — Rules

Rules define how code should be written.

Example:

HTML rules
CSS rules
animation rules

---

# Layer 2 — Patterns

Patterns define page structure.

Examples:

landing page
local business website
SaaS landing

---

# Layer 3 — Components

Components define reusable UI sections.

Examples:

hero
features
testimonials
CTA

---

# Layer 4 — Templates

Templates define generation logic.

They combine patterns and components.

---

# Layer 5 — Presets

Presets define visual style.

Examples:

luxury
minimal
dark
