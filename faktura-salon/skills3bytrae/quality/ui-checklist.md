# UI/Design Checklist

Ensure the site looks "Premium" and not broken.

## Spacing & Alignment
- [ ] **Consistency**: Are margins/paddings using the spacing scale?
- [ ] **Alignment**: Are items in grids aligned correctly?
- [ ] **Breathing Room**: Is there enough white space?

## Typography
- [ ] **Hierarchy**: Is H1 clearly bigger than H2?
- [ ] **Readability**: Is body text contrast high enough? Line height comfortable (1.5)?
- [ ] **Font Loading**: Do fonts load without layout shift?

## Interaction
- [ ] **Hover States**: Do buttons/links change on hover?
- [ ] **Focus States**: Are inputs highlighted when clicked?
- [ ] **Mobile Menu**: Does it open/close smoothly?
