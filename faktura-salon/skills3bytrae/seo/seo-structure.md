# SEO Structure & Heading Hierarchy

Correct heading structure is vital for accessibility and SEO.

## Rules

1.  **One H1 per page**: This is the main title. Must contain the primary keyword.
2.  **H2 for Sections**: Each major section (Services, About, Testimonials) starts with an H2.
3.  **H3 for Cards**: Items inside a section (Service Name, Reviewer Name) use H3.
4.  **H4-H6**: Use sparingly for sub-details.

## Example Structure (Landing Page)

-   **H1**: Best Plumbing Services in Chicago
    -   **H2**: Emergency Repairs 24/7
        -   **H3**: Leak Detection
        -   **H3**: Pipe Burst Repair
    -   **H2**: Why Choose Us?
        -   **H3**: Licensed Experts
        -   **H3**: Fair Pricing
    -   **H2**: Customer Reviews
        -   **H3**: "Saved my basement!" - John D.
    -   **H2**: Contact Us

## Common Mistakes

-   Using H2s for styling (e.g., making text big) instead of structure.
-   Skipping levels (e.g., H1 -> H4).
-   Using H1 in the logo (unless it's the home page, but text is better).
