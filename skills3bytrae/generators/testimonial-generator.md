# Testimonial Generator

Use this prompt to generate realistic social proof.

## Prompt

```markdown
Generate 3 realistic testimonials for a [Niche] business in [City].

**Personas:**
1. **The Busy Parent**: Appreciates speed and convenience.
2. **The Skeptic**: Was worried about price/quality but was pleasantly surprised.
3. **The Local**: Loves supporting local business.

**Format:**
- Quote: 2-3 sentences. Specific details (not generic).
- Name: First Name + Last Initial.
- Location: [City] resident.
- Rating: 5 Stars.

**Example:**
"I was worried about the cost, but [Business Name] was upfront and honest. No hidden fees, and the work was done in half the time I expected." - Mark S., Austin
```
