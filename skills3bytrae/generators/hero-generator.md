# Hero Copy Generator

Use this prompt to generate high-converting hero section copy.

## Prompt

```markdown
Act as a direct-response copywriter. Write content for the Hero Section of a [Niche] website in [City].

**Requirements:**
1. **Headline (H1)**: 
   - Must be benefit-driven. 
   - Must include the location ([City]).
   - Max 10 words.
2. **Subheadline**: 
   - Address the main pain point.
   - Establish authority (e.g., "Rated #1", "20 years experience").
   - Max 25 words.
3. **Primary CTA**: 
   - Action-oriented (e.g., "Book Now", "Get Quote").
   - High contrast.
4. **Trust Badge**:
   - A short phrase to go near the button (e.g., "No credit card required", "Licensed & Insured").

**Banned Words**: seamless, cutting-edge, innovative, premier.
```
