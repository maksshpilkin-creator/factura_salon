# AI Operating System Instructions

This `skills/` folder contains all the context needed to build high-quality local business landing pages.

## Quick Start

1.  **Start a New Project**:
    -   Go to `skills/flow/landing-page-flow.md`.
    -   Follow the step-by-step guide.

2.  **Add a Feature**:
    -   Go to `skills/flow/feature-development-flow.md`.
    -   Example: Adding a booking form or gallery.

## Folder Overview

-   **`components/`**: Copy-paste HTML/CSS for Heroes, Navbars, Footers.
-   **`design/`**: Copy the CSS variables from `design-system.md` into your `styles.css`.
-   **`generators/`**: Use these prompts to generate content (text, FAQs).
-   **`niches/`**: Read the specific niche file (e.g., `dentist.md`) for color/style rules.
-   **`patterns/`**: Choose a layout pattern (e.g., `landing-page.md`) to structure your HTML.
-   **`presets/`**: Apply a visual theme (e.g., `luxury.md`) by updating CSS variables.
-   **`quality/`**: Run through these checklists before delivery.
-   **`rules/`**: The "Laws" of the project. AI must follow these.
-   **`seo/`**: Templates for meta tags and structured data.
-   **`templates/`**: Full starter HTML files.
