      Каждая иллюстрация — отдельный React-компонент в components/illustrations/.
      Реэкспорт через index.ts.

      Технические требования:
      — viewBox="0 0 400 260", fill="none", className="h-full w-full"
      — Цвета ТОЛЬКО через linearGradient с CSS-переменными:
        stop stopColor="var(--color-primary)" и stop stopColor="var(--color-accent)"
        Белый (white) для акцентных деталей.
      — ID градиента уникальный для каждой иллюстрации (il-speed, il-shield и т.д.).
      — Без text, image, foreignObject, clipPath, mask, filter — только базовые SVG-примитивы.

      Визуальный стиль:
      — Центральный объект крупный и считываемый — человек должен за 1 секунду понять тему.
      — 3-4 слоя глубины с убывающей opacity (0.03 → 0.9).
      — Тонкие линии (strokeWidth 0.5-2), полупрозрачные заливки (fillOpacity 0.03-0.15).
      — Декоративные элементы вокруг: орбитальные круги, пунктирные линии, точки, дуги.
      — Один акцентный элемент с opacity 0.8-1.0 (белая точка, галочка, курсор), остальное приглушено.
      — Стиль как у Linear, Raycast, Vercel — абстрактный, но считываемый.
      — НЕ примитивные квадраты и прямоугольники. Используй кривые (path с Q/T), многоугольники, комбинации.
    </svg_illustrations>
